What’s this?

The fiëé croisée project aims to become a semi-automatic crossword puzzle generator. Even if there's still a long way to go, it’s already a valuable help in creating crossword puzzles.

Proceed to the puzzle grid page and create a new puzzle. At the moment you can print the page from the browser or export complete layout data for LaTEX (cwpuzzle) or ConTEXt. Soon we can also provide print-ready puzzles as PDF.

More details come with the source code at GitHub.
